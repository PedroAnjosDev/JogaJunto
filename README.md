# Sistema de Campeonato de Futebol

## Descrição

Este projeto consiste em um sistema simples de gerenciamento de campeonato de futebol, desenvolvido em Java, utilizando Programação Orientada a Objetos (POO).

O sistema permite cadastrar equipes, registrar partidas e organizar competições.

---

## Funcionalidades

* Cadastro de equipes
* Cadastro de partidas
* Associação de equipes às partidas
* Registro de placares
* Organização por competição

---

## Estrutura do Projeto

O projeto está organizado em pacotes:

* `entidade`

  * Classe `Equipe`
  * Classe `Partida`
  * Classe `Competicao`

* `main`

  * Classe principal para execução

---

## Tecnologias Utilizadas

* Java
* Eclipse IDE

---

## Como Executar

1. Importar a pasta do projeto no Eclipse
2. Localizar a classe principal (Main/App)
3. Executar como Java Application

---

## Observações

* O projeto utiliza listas para armazenar dados em memória
* Não há integração com banco de dados
